# smart-cv-filtering

AI-powered CV filtering and screening platform designed to help HR teams
automatically evaluate and shortlist candidates.

## Project Overview
This project uses machine learning techniques integrated with a Django backend
to analyze CVs and provide intelligent recommendations for HR decision-making.

## Features
- Upload and parse CVs (PDF)
- AI-based CV classification and filtering
- HR dashboard API
- Candidate scoring and shortlisting
- RESTful API architecture

## Tech Stack
- Django & Django REST Framework
- Python
- Machine Learning (NLP-based CV analysis)
- PostgreSQL
- REST API

## Installation
1. Clone the repository
2. Create virtual environment
3. Install dependencies
